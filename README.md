# 项目简介

## 核心功能
- **便捷点餐**：用户可快速浏览菜品，选择心仪菜品进行下单。
- **AI问答**：通过智能问答系统获取菜品推荐或服务咨询。
- **购物车管理**：用户可添加或删除菜品，调整数量，查看购物车详情。
- **订单查询**：用户可查看订单状态，包括已下单、配送中、已完成等。
- **地址管理**：用户可添加、编辑、删除收货地址。
- **收藏菜品**：用户可收藏常用菜品，方便下次快速下单。
- **反馈入口**：用户可提交建议，帮助优化小程序功能。
- **个人中心**：用户可查看与管理个人相关信息，包括订单历史、收藏菜品等。

## 适用场景
- 校园食堂：适用于校园内的食堂，为师生提供便捷的点餐服务。
- 餐饮行业：适用于各类餐饮企业，提高点餐效率和用户体验。

## 技术特点
- 基于SpringBoot框架，保证系统的稳定性和可扩展性。
- 使用Uniapp开发，实现跨平台应用，方便用户在不同设备上使用。
- 集成AI问答功能，提升用户互动体验。
- 界面简洁直观，提高用户操作便捷性。
